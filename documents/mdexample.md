# Another Markdown Example

Here's another markdown file with links.

### Good Links

- [GitHub](https://github.com)
- [Wikipedia](https://www.wikipedia.org)

### Bad Links

- [Imaginary Site](https://www.google.com/404)
- [link with whitespace](https://translate.google.com/%20404)
- [Google Search Section (fragment)](https://www.google.com/search#about)
- [url with query](https://www.google.com/search#about)
- [Bad Path](https://www.github.com/thispagedoesnotexist)
- [Duplicate](https://www.github.com/thispagedoesnotexist)
- [trailing hash](https://example.com/page?query=2#/)
